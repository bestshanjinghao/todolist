# 信用卡活动管理系统

一个用于管理信用卡活动的 Web 应用，支持活动管理、提醒设置、数据统计等功能。

## 功能特点

- 📝 活动管理：添加、编辑、删除信用卡活动
- 🏦 银行管理：管理不同银行的活动
- ⏰ 提醒功能：设置活动开始和截止提醒
- 📊 数据统计：活动数据可视化展示
- 📅 日历视图：直观展示活动时间分布
- 📤 数据导出：支持导出活动数据

## 技术栈

- **框架**: Next.js 14 (App Router)
- **UI 库**: Ant Design
- **数据库**: SQLite (通过 Prisma)
- **图表**: Recharts
- **状态管理**: React Hooks
